# hashed-timelock-contract-dapp
A dAPP for hashed timelock contract

## Environment
`Truffle v5.5.4`  
`Solidity: 0.8.12`
`Ganache v7.0.3`
`Node v17.7.1`

## Usuage

### Install
`npm install`

### Start test network
`npm run ganache-start`

### Run contract unit tests
`npm run test-contracts`
